# AIGC Token 预算计算器

一个纯前端单文件（`token-budget.html`）的网页工具，用于测算编辑器内 **AIGC 能力的成本预算**：底层按百万 token 计费，表现层对用户呈现为「单次生成」次数，并按段位拆解预算占用、给出次数付费的回本定价建议。

## 功能

- **底层 Token 模型**：按「元 / 1M tokens」输入单价，支持 1~100% 折扣系数，自动折算折后单价与单次生成成本。
- **用量假设（按段位）**：每个段位可单独填写「人均次/天」，也支持一键批量设置；免费额度从用量中扣除，仅超出部分计费。
- **统计口径切换**：可在「周活人数」与「当前段位人数」之间切换，当前生效的口径列在表格中高亮标注，二者不叠加。
- **各段位预算明细**：展示当前人数、周活人数、月计费次数、月成本及占总预算比例，并自动提示预算消耗最高的段位。
- **每日 / 每周 / 每月预算**：实时联动计算。
- **次数付费定价 & 回本分析**：支持单次售卖与打包售卖两种模式，计算毛利、毛利率、是否回本，并给出保本价 / 30% / 50% 毛利的定价建议。

## 使用

直接用浏览器打开 `token-budget.html` 即可，所有计算在本地浏览器完成，修改任意输入即时刷新。

本地起服务预览（可选）：

```bash
python -m http.server 8765
# 然后访问 http://localhost:8765/token-budget.html
```
